# Teslo API

1. Clonar proyecto
```bash
git clone
```

2. ```yarn install```

3. Clonar archivo ```.env.template ``` y renombrar a ```.env```

4. Cambiar variables de entorno

5. Levantar base de datos
```bash
docker-compose -up -d
```

6. ```yarn start:dev```
